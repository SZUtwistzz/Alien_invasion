# Alien Invasion (外星人入侵)

这是一个基于 Python 和 Pygame 开发的经典街机射击游戏《外星人入侵》。玩家控制一艘飞船，通过射击消灭不断下沉的外星舰队，同时躲避外星人的碰撞和子弹。

## 游戏特色

*   **经典玩法**：控制飞船左右移动，发射子弹消灭外星人。
*   **外星人反击**：外星人不再只是移动，它们会随机向飞船发射红色子弹，玩家需要灵活躲避。
*   **动态难度**：每消灭一群外星人，游戏等级提升，外星人移动速度加快，挑战升级。
*   **记分系统**：包含实时得分、等级显示以及最高分记录。
*   **最高分持久化**：游戏关闭后会自动保存最高分，下次启动时读取。
*   **音效支持**：包含射击和爆炸音效（需要在 `sounds/` 目录下放入 `bullet.wav` 和 `explosion.wav`）。
*   **生命值系统**：玩家有 3 次机会（飞船），被外星人撞击、被外星人子弹击中或外星人到达屏幕底部都会损失一艘飞船。

## 环境要求

*   Python 3.x
*   Pygame

## 安装与运行

1.  **克隆或下载本项目** 到本地。

2.  **安装依赖**：
    如果尚未安装 Pygame，请运行以下命令：
    ```bash
    pip install pygame
    ```

3.  **准备资源**：
    *   确保 `images/` 文件夹下有 `ship.bmp` (飞船) 和 `alien.bmp` (外星人) 图片。
    *   (可选) 确保 `sounds/` 文件夹下有 `bullet.wav` (射击音效) 和 `explosion.wav` (爆炸音效) 以启用声音。

4.  **运行游戏**：
    在项目根目录下运行主程序：
    ```bash
    python alien_invasion.py
    ```

## 操作说明

*   **左右方向键 (← / →)**：控制飞船左右移动。
*   **空格键 (Space)**：发射子弹。
*   **Q 键**：退出游戏（同时保存最高分）。
*   **Play 按钮**：点击屏幕上的 "Play" 按钮开始新游戏。

## 文件结构

*   `alien_invasion.py`: 游戏主程序入口，管理游戏资源和主循环。
*   `settings.py`: 包含游戏的所有设置类（屏幕大小、速度、分数等）。
*   `game_stats.py`: 跟踪游戏的统计信息（得分、等级、剩余飞船）。
*   `scoreboard.py`: 负责在屏幕上显示得分信息。
*   `ship.py`: 管理玩家飞船的行为。
*   `bullet.py`: 管理子弹的行为。
*   `alien.py`: 管理外星人的行为。
*   `button.py`: 自定义按钮类，用于创建 "Play" 按钮。
*   `high_score.txt`: 用于存储历史最高分的文件。
*   `images/`: 存放游戏图片资源。
*   `sounds/`: 存放游戏音频资源。
